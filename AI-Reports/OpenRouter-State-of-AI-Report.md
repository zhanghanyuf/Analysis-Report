# **1. 报告背景与数据来源**

### **数据规模**

* 基于 **100+ 万亿 tokens**（真实推理请求）
* 覆盖 **300+ 模型**、**60+ 模型提供方**
* 来源于全球用户真实调用，而非 benchmark 或实验

### **数据特点**

* 无用户内容（仅 metadata）
* 分类基于 GoogleTagClassifier
* 时间跨度约 2 年，重点为过去 13 个月

### **重要性**

这是目前市场上 **最接近真实商业使用情况** 的 LLM 使用报告，是了解“用户实际怎么用 AI”的最佳窗口。

---

# **2. 开源 vs 闭源：市场格局变化**

### **整体结论**

* 闭源（OpenAI/Anthropic/Google 等）仍是主导
* 但 **开源迅速崛起到 30% 市场占比**
* 增长主要来自：

  * DeepSeek（中国产）
  * Qwen（阿里）
  * Moonshot
  * GPT-OSS
  * MiniMax

### **趋势拐点**

* 每次开源发布（DS V3、Qwen3、GPT-OSS）都会带来使用激增
* 且使用激增不是短暂试用，而是 **长期留存**

### **推论**

* 技术差距在快速缩小
* 开源开始具备“商业可用性”
* 多模型组合（multi-model routing）正在成为企业标配

---

# **3. 开源生态内部竞争格局**

### **2024：DeepSeek 一家独大**

* DeepSeek V3 和 R1 占据超 50% 开源流量
* 形成明显“头部垄断”

### **2025：格局碎片化**

* Qwen、Moonshot、MiniMax、GPT-OSS 快速崛起
* 到 2025 Q4：

  * 无单一模型占比超过 20–25%
  * 出现五至七家中大型玩家

### **启示**

* 开源模型具备 **快速打平差距** 的能力
* 用户迁移成本极低：只要模型更好/更便宜 → 会直接切换
* 开源竞争是“日更级速度” vs 闭源的季度迭代

---

# **4. 模型规模趋势（Small / Medium / Large）**

| 规模           | 参数量     | 趋势         | 用户偏好                   |
| ------------ | ------- | ---------- | ---------------------- |
| ① 小模型 < 15B  | 数量多、碎片化 | **下降趋势**   | 适合本地推理、边缘端             |
| ② 中模型 15–70B | 爆发增长    | **最强势增长段** | 价格低 / 能力强 / 速度快（最佳平衡点） |
| ③ 大模型 ≥ 70B  | 多家竞争    | **稳定增长**   | 用于复杂任务、推理、企业级服务        |

**中型模型成为“新主流”，取代大量小模型的应用场景。**

代表模型：Qwen2.5 32B、Mistral Small 3、GPT-OSS-20B 等。

# **5. LLM 使用类别分析**

整体来看，用户真实使用主要集中在两类：

## **① Roleplay（角色扮演）≈ 50%+（OSS 模型中）**

* 在开源模型中占比超过一半
* 原因：

  * 内容更自由
  * 社区生态强（情感陪伴、游戏、故事、persona）
  * 创意写作成本低
* Roleplay 场景高度黏性

## **② Programming（编程）占比暴涨，从 11% → 50%+**

* 这是全报告中最震撼的趋势
* 2025 已进入“AI 主导编程时代”

关键驱动因素：

* GPT-5、Claude 4.5、Gemini 2.5 的 reasoning + tool
* IDE 深度融合
* LLM 成为代码调试、生成、搜索的主力

### **其他类别**

* Translation（翻译）
* General Q&A
* Productivity
* Education
* Health / Legal（类别分布分散）

**高价值任务集中在“编程 + 推理 + 多步操作”。**

# **6. Agentic Inference 的崛起（2025 最重要趋势）**

Agentic inference = 多步推理 + 工具链调用 + 长上下文 + 状态保持

OpenRouter 数据表明：

### **① Reasoning models 占比 > 50% 全部 tokens**

Top models:

* Grok Code Fast
* Gemini 2.5 Pro / Flash
* GPT-OSS-120B
* Claude 4.5 Sonnet

### **② 工具调用（tool calling）持续增长**

* 用户不再是“一问一答”
* 而是使用模型执行复杂任务链路

### **③ Prompt 长度 ×4**

→ 用户塞更大的任务（代码 / 文档 / 数据）

### **④ Completion 长度 ×3**

→ 模型推理链路加长

### **⑤ Sequence length 跳到 5000+ tokens**

→ 高度复杂任务成为主流

**结论：LLM 不再是聊天机器人，而是“自动化执行器”。**

# **8. 用户留存：Glass Slipper（玻璃鞋）效应**

核心观点：

* 大多数用户短期试用后流失
* 但 **找到“合适模型”的用户留存极高**
* 一旦某个模型解决了关键任务 → 长期绑定
* 这种绑定不是情感，而是业务流程与数据管线绑定

这解释了为什么：

* DeepSeek V3 的留存曲线很漂亮
* Claude、Gemini、OpenAI 的技术模型留存持续上升

# **9. 结论（面向企业/创业者）**

核心洞察可以压缩为五条：

1. **多模型时代已确定**（单模型不够）
2. **中型模型才是主流商业规模**
3. **AI 编程将重塑所有行业**
4. **Agentic 是未来 2 年主战场**
5. **Roleplay 为内容与消费类 AI 提供巨大空间**
